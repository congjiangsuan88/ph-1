# 🇵🇭 PH-1 在菲生活助手 (PH-1 Life Helper)

> **访问地址：** [https://ph-1.com](https://ph-1.com) 

### 🌟 项目简介
PH-1 是一款轻量级、响应式的 Web 工具站，旨在解决在菲华人高频的生活痛点。项目采用纯前端架构，确保在菲律宾移动网络环境下极速加载。

### 🛠️ 核心功能
- **双向汇率换算**：接入实时 API，自动计算官方价与市场钱庄参考价。
- **签证到期管家**：支持旅游签、9G 工签等多种类型的续签提醒，并提供日历闹钟下载。
- **办事处指南**：收录移民局(BI)、外交部(DFA)、NBI 等核心部门的地址、Grab 预估价及独家避坑贴士。
- **SEO 优化**：深度适配 Google 搜索，针对菲律宾本地关键词进行语义增强。

### 🚀 技术栈
- **框架**：Tailwind CSS (响应式布局)
- **图标**：SVG Inline Icons (零延迟加载)
- **部署**：Vercel / GitHub Pages
- **数据**：ExchangeRate-API (实时汇率)
