隐私政策

更新日期：2026年3月

1. 数据收集

PH-1 菲华助手（以下简称“本站”）是一个纯前端工具。我们**不要求**用户注册，也不会在服务器上存储您的任何个人财务输入（如换汇金额、签证日期）。

2. 本地存储 (LocalStorage)

为了提升用户体验，本站会使用浏览器的本地存储功能缓存汇率数据。这些数据仅保存在您的个人设备上，不会上传至我们的服务器。

3. 第三方服务

我们使用第三方 API (ExchangeRate-API) 获取实时汇率，并使用 CDN 加载静态资源。这些服务可能会收集您的 IP 地址以提供基础的网络连接服务。

4. 联系我们
